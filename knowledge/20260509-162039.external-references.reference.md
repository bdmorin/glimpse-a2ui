## Worked-Example / Adjacent Integrations

| What | URL | Notes |
|------|-----|-------|
| OpenClaw Canvas docs | `https://docs.openclaw.ai/platforms/mac/canvas.md` | Third-party integration of A2UI in a desktop agent client. | OpenClaw + A2UI blog | `https://launchyourclaw.com/blog/visualizing-work-with-openclaw-canvas-and-a2ui` | High-level walkthrough. Confirms A2UI-as-protocol vs Canvas-as-app-feature distinction. |

## Things That Look Adjacent But Aren't

- **Anthropic Claude.ai Artifacts.** Server-rendered HTML in the chat UI. Closed, not a protocol, not relevant to local-agent UI.
- **MCP (Model Context Protocol).** Tool-call protocol, not a UI protocol. **Pairs with** A2UI/glimpse-a2ui as the agent-side tool surface; doesn't replace it.

## How to Use This File

When a future session asks "where do I find the A2UI spec?" — point them here, don't re-search. Update this file when new authoritative sources appear or old ones rot.
